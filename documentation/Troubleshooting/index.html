    VSCode in WSL

    “Missing .Net SDK” message when executing code . in WSL

    The Omnisharp plugin for VSCode may have trouble finding the dotnet setup locally for the MLOS repo in tools/, even if you source the scripts/init.linux.sh script to setup your PATH environment.

    To workaround this issue, you can install dotnet system wide for your WSL2 distro.

    Here are the instructions for Ubuntu 20.04:
    https://docs.microsoft.com/en-us/dotnet/core/install/linux-ubuntu#2004-
