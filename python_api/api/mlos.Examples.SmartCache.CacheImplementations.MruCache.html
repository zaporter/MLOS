mlos.Examples.SmartCache.CacheImplementations.MruCache¶

class mlos.Examples.SmartCache.CacheImplementations.MruCache.MruCache(max_size, logger)[source]¶

Bases: mlos.Examples.SmartCache.CacheImplementations.XruCache.XruCache

An implementation of a Most Recently Used (MRU) cache.

Supposedly this type of cache works well in case of: 1) Random accesses 2) Cycling through the dataset that cannot fit in the cache

The idea being that the longer something has been in the cache, the higher the probability that it will be used again (at least in case of 2).
