Parameter to sklearn lasso regressor controlling how model coefficients are selected for update. From https://scikit-learn.org/stable/modules/generated/sklearn.linear_model.Lasso.html: If set to ‘random’, a random coefficient is updated every iteration rather than looping over features sequentially by default. This (setting to ‘random’) often leads to significantly faster convergence especially when tol is higher than 1e-4.
